SequenceDistanceGraph¶

class SequenceDistanceGraph : public DistanceGraph¶

Class representing sequence graphs.

This class represents the assembly problem as a flow formulation. The most important features of this class are:

  • Nodes start from 1.

  • Nodes are signed to represent the direction in which they are being considered.

  • Nodes are saved in canonical form.
